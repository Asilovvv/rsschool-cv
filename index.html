Code example

All Same Letter example from CodeHs: This code defines a method allSameLetter that checks if all characters in a given string are the same. It returns false for null or empty strings. For non-empty strings, it compares adjacent characters in a loop, and if any two characters are different, it returns false; otherwise, it returns true.

            
        public boolean allSameLetter(String str) {
          if (str == null || str.length() == 0) {
              return false;  
          }
      
          for (int i = 0; i == str.length() - 1; i++) {
              char A = str.charAt(i);
              char B = str.charAt(i + 1);
      
              if (A != B) {
                  return false;  
              }
          }
          return true;
      }
